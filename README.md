# 🍬 Word Crush

**Word Crush** is a Flutter-based Turkish word puzzle mobile game developed for **Software Laboratory II**.  
Players create Turkish words by dragging across neighboring letter tiles on a grid. Valid words are checked using a local Turkish dictionary and scored according to letter values. The game also includes combo bonuses, special power tiles, joker boosters, a market system, and score history tracking.

## 🎮 Features

- Turkish word puzzle gameplay
- Drag-based letter selection
- Easy, Medium, and Hard difficulty levels
- Custom move selection before starting the game
- Turkish dictionary-based word validation
- Letter score calculation
- Combo scoring with subword detection
- Special power tiles:
  - Row Clear
  - Column Clear
  - Bomb
  - Mega
- Joker boosters:
  - Fish
  - Wheel
  - Lollipop
  - Swap
  - Shuffle
  - Party
- Joker market system using in-game gold
- Animated tile effects and joker animations
- Score history and statistics
- Local data storage for username, gold, jokers, and game records
